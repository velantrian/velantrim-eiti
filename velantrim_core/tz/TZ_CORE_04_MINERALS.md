## ✅ Обязательные поля для каждого минерала

```
formal_notation ОБЯЗАТЕЛЬНО содержит:
  - Химическая формула
  - Сингония (кристаллическая система)
  - Плотность ρ г/см³
  - Твёрдость по Моосу
  - Блеск (стеклянный/металлический/перламутровый/жирный/шелковистый)
  - Цвет (типичный)
  - Спайность (совершенная/несовершенная/отсутствует) если важна

conditions: условия стабильности, агрегатное состояние

tags: класс минерала + типичное применение
```

## ✅ Критерий отбора топ-500

```
Брать минерал если выполняется хотя бы одно:
  ✅ Породообразующий (>1% земной коры)
  ✅ Промышленно важный (руда, флюс, абразив, оптика)
  ✅ Биологически важный (апатит в костях, магнетит в нейронах)
  ✅ Часто упоминается в геологии/петрографии
  ✅ Драгоценный или полудрагоценный (ювелирная ценность)
  ✅ Токсичный (асбест, арсенопирит — знать чтобы избегать)
```
